# What is a "kalefetch"?

Hi, I'm Kale. Ever since I was little, I've been obsessed with tech, and that's what leads me here today.

But what *is* a "kalefetch"?

The name comes from a mix of two things. "Kale" is just a nickname I started using online, and "fetch" comes from the world of Linux.

If you've ever used tools like ```neofetch``` or ```fastfetch```, you know they print system information in the terminal along with some ASCII art.

So "kalefetch" is kind of my own version of that idea: a small snapshot of who I am on the internet.
